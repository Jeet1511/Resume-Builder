## Tech Stack
- **Frontend**: React 18 + Vite, React Router, Axios, html2pdf.js
- **Backend**: Node.js, Express, MongoDB/Mongoose, JWT Authentication
- **Styling**: Custom CSS with glassmorphism, dark theme, animations

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Opens at `http://localhost:5173`

### Backend
```bash
cd backend
npm install
# Edit .env with your MongoDB URI
npm run dev
```
Runs API on `http://localhost:5000`
